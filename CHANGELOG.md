# Changelog

## 0.1.0-alpha.1 - 2026-07-20

- Initial public Exasol Gateway SDK extraction.
- Stable `sessiongw_c_*` C ABI and typed C++ facade.
- SGW1 capability negotiation, metadata, cursor, transaction, native read, and
  native mutation support.
- Additive `ExasolGateway::Sdk` and `ExasolGateway::C` CMake targets with
  SessionGateway compatibility aliases.
